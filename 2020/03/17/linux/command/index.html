  • date日期

  • cal日历
    cal month year

  • pwd显示当前目录（路径）

  • ls显示当前目录中的文件和目录
    ls -l

  • mkdir创建空目录（只能一层一层来）
    要想创建多层的目录要加选项-p

  • rmdir删除目录(要为空)

  • rm -rf删除目录（可以不为空）

  • wc查看单词数量

  • cp <file1> <directory/file2> 将file1文件复制到目录，还能更换成file2这个名字

  • cp -r <dir1> <dir2> 将目录1复制到目录2

  • cp -s <file1> <firectory/file2> 复制为符号链接

  • cp -l <file1> <firectory/file2> 复制为硬链接

  • cat 查看文件信息

  • less 查看文件信息

  1. 搜索输入的字符/<str>
  2. 支持上下翻页

  • head tail 查看前面10行，后面10行
    -n number显示number行

  • nl 显示文件内容，同时打印行号

  • diff 检查文件的不同

  • head <file> -n 2查看文件的前两行

  • tail <file> -n 2查看文件的后两行

  • rm删除文件

  • mv移动文件，重命名文件
    mv <file1> <folder> //移动
    mv <file> <File> //重命名

  • touch创建一个不存在的文件，或者更新（修改，读取）文件的时间

  • ll --time=atime查看文件读取时间
    ll --time=ctime查看文件权限更改时间
    ll --time=mtime查看文件修改时间
    （ll是ls -l的缩写）

  • chmod权限管理
    chmod <ugo-rwx> <file>
    关于权限:
    输入ls -l可以看到，开头的（－）（ｄ）是表示（文件）（文件夹）
    －ｒｗｘ｜ｒ－－｜ｒ－－｜　　　　／／作者（ｕ）｜小组（ｇ）｜其他人（ｏ）
    ｒ　可读
    ｗ　可写
    ｘ　可执行
    ＋　添加权限
    －　去除权限

  • chown改变文件/目录拥有者

  • chgrp改变文件/目录加入的用户组
