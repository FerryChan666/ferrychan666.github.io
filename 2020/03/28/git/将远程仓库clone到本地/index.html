Created2020-03-28|Updated2022-05-22|github
|Post View:

好处

当一起合作开发项目的时候，就可以直接通过到GitHub上面，将仓库clone到本地，不需要在本创建仓库了。
clone有两种传输协议。

  1. HTTPS
    这种协议不用SSH　KEY，方便直接。但clone起来会较慢。
  2. SSH
    需要创建KEY，特点会更快。
