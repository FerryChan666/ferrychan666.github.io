verbs with and without object

After some verbs we typically or always add a completion - a phrase which completes the meaning of the verb - which can be an adverb or preposition phrase.

  1. Some verbs are followed by an object. (There are called transitive verbs)

    • Some verbs which are typically or always followed by a completion are transitive in there most common meanings.

    • Include base…on, compare…with, interest…in, lend … to, prevent…from, regard…as, remind…of,supply…with

  2. Some verbs are not followed by an object. (There are called intransitive verbs)

    • Some verbs which are typically or always followed by a completion are intransitive in there most common meanings.

  3. Some verbs can be both transitive and intransitive.

verb + to infinitive or bare infinitive

verb + (object) + infinitive

  1. After some verbs, we need to include an object before a to-infinitive

  2. After some verbs, we can’t include an object before a to-infinitive.

verb + (object) + bare infinitive

  1. make, feel, notice, observe, overheard, see, watch, have, let.

Notice, however, that in passive sentences with these verbs, we use a to-infinitive

verb + wh-clause

  1. Some verbs can be followed by a clause beginning with a wh-word(how, what, when, where, which, who or why, whether)

    • These verbs can also be followed by wh-word(expect ‘why’) + to-infinitive

  1. Some verbs must have an object before the wh-clause

    • These verbs can also be followed by wh-word + to-infinitive

  1. whether

    Whether can be followed by wh-word + to-infinitive, but if can’t.

verb + two objects

Some verbs are followed by two objects. Usually the first object is a person and the second object is a thing.

  1. We can reverse the order of the objects if we put the for or to before object1. (this is called a prepositional object)

    • we often use this patter if we want to focus particular attention on the object after for/to

    • we also use it if object1 is a lot longer than object2.

  • Sometime we use for + object with verbs such as book, build, buy, catch, choose, cook, fetch, find, make, order, pour, save.

  • Sometime we use to + object with verbs such as award, give, hand, lend, offer, owe, pass, show, teach, tell, throw.

  • With some other verbs we can use either to or for, including bring, leave, pay, play, post, read, send, sing, take, tell, throw.(There is very little difference in meaning.)
