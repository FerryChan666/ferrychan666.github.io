Preface

Canvas is a container for graphics. You must use JavaScript to draw the graphics.

Draw on the canvas with JavaScript

  1. Get the canvas

    We can get it by using the JS method document.getElementById("")

  2. set an object from canvas

    var ctx=c.getContext("2d");

    This means the drawing object draws with 2 dimension.

  3. draw the graph by using the object

    We can use this drawing object to draw the graphics with moveTo(x,y), lineTo(x,y) and arc(x, y, r, startangle, endangle) method.

    And Then, it is imperative for us to add the ink by using stroke()method.
