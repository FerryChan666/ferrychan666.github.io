库：urllib, requests

purllib是Python 内置的HTTP 请求库，包含四部分

  1. urllib.request （打开和读取URL）
  2. urllib.error (包含 urllib.request 抛出的异常）
  3. urllib.parse （用于解析URL）
  4. urllib.robotparser （ 用于解析robots.txt 文件）

异步爬虫

动态网页

对于动态网页，从我们加载网页到整个页面呈现出来之后，不只是经过了一次请求，有的还通过了JavaScript的渲染，增加，修改基本的html信息。那么我们使用requests请求网页，只是返回了基本的response，里面包含的只是基本的html数据，网页中通过Ajax请求（异步加载），通过requests返回的response并没有包含通过JS渲染过后的html数据。

为爬取这种动态的网页，我们虽然可以找到Ajax请求的url，但由于有的网页的审查过与严格，通常会无法访问。所以，我们可以使用

  • selenium

    模拟浏览器，来实现自动化操作，达到爬取网站的目的
