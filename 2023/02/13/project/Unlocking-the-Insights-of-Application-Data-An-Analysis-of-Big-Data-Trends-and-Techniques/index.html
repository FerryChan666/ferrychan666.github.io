Model Building

  • Explain the regression analysis technique you used to analyze the data.
  • Discuss how you selected the features and target variables.
  • Highlight the limitations of the model and how you overcame them.

Simple linear regression analysis is a technique used to understand the relationship between two continuous variables, where one variable (dependent) is considered to be an outcome or response and the other variable (independent) is considered to be a predictor. In the analysis, we try to establish a linear relationship between these two variables.

In the data analysis process, the first step is to select the features and target variables. For this analysis, we selected the independent variable as “Work experience” and the dependent variable as “Salary.” The selection of the independent and dependent variables is based on the assumptions that the work experience has a linear relationship with the salary, and it makes sense to predict the salary based on the work experience.

After pre-processing the data, we fit the simple linear regression model using the work experience and salary data. The regression line represents the best-fit line for the data, and the slope and intercept of the line are the coefficients of the regression equation.

Model Evaluation

  • Evaluate the performance of the model by using various metrics such as R-squared, Mean Squared Error, etc.
  • Discuss the results of the evaluation and what they mean for the analysis.
  • Provide an interpretation of the results and the implications for the analysis.

In evaluating the performance of the simple linear regression model, various metrics can be used to assess the accuracy of the model’s predictions. One commonly used metric is R-squared, which measures the proportion of the variance in the target variable that can be explained by the features. A higher R-squared value indicates that the model is better at explaining the variance in the target variable. Another metric used to evaluate the performance of the model is the Mean Squared Error (MSE), which measures the average of the squared differences between the actual and predicted target values. A lower MSE value indicates that the model’s predictions are more accurate.

result

The R-Squared value measures the proportion of variance in the dependent variable (salary) that is explained by the independent variable (work experience). An R-Squared value of 0.08 means that only 8% of the variance in salary is explained by work experience.

The Mean Squared Error (MSE) is a measure of the average squared difference between the actual target values and the predicted target values. An MSE value of 108.83 means that, on average, the model’s predictions are off by 108.83 units of the target variable (salary).

In general, a low R-squared value combined with a high MSE value indicates that the model is not a good fit for the data. There is low relationship between experience of work and salary. The key findings of the analysis suggest that experience of work alone cannot be used as a strong predictor of salary.
