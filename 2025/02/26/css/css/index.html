CSS rule

A CSS rule consists of a selector and a declaration block.

Selector

  1. element selector

    The element selector selects elements based on the element name.

    Now all the h1 will be center-aligned, with while color.

  2. id selector

    The id selector uses the id attribute of an element to select an element.

  3. class selector

    The class selector selects elements with a special class attribute.

Insert CSS

There are three ways of inserting css.

  • external css

    Each HTML page must include a reference to the external style sheet file inside the <link> element, inside the head section.

  • Inline css
