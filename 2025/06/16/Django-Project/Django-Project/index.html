Extends html

After we wrote the HTML file above, we found that there are too much repetitive code.

In the Django, we can reduce these repetitive code by using the inherit method.

  1. base.html

    We define the HTML code that every html file needs in the base.html, such as the head, header and so on.

    Then, we use {% block name %}{% endblock %} as a placeholder to be replaced by other HTML files that inherit the base.html.
