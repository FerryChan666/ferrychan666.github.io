并列句

  1. or, not only…, but also… (and)

从句

定语从句

本从句需要引导词充当成分。

  1. 人+ who(人充当主语）, whom（人充当宾语）,whose
  2. 物+ that, which
  3. 时间+ when
  4. 地点+where
  5. As …, …（As 指代, 后面整句话
    • As is symbolically illustrated in the cartoon, a person …

状语从句

  1. when（当）
  2. Once …, … (一旦)
  3. As…, …(由于)
  4. in that = because, since, as, for(因为（强度由高到低)
  5. while, through(尽管)

宾语从句

  1. 动词/介词 + that, what

主语从句

It is … that…

同位语从句

一般用于抽象名词后,

  • I cling to the idea that …

非谓语动词

动名词

doing将动词+ing变成名词

成分：主语和宾语

  • [Participating] in volunteer works could boost my development of interpersonal relationship.

Participating充当主语

不定式

to do

成分：能充当任何成分

  • It is necessary for us [to use computer in a reasonable way and restrain from overindulgence].

It 为形式主语，真正的主语是不定式，to use computer in a reasonable way and restrain from overindulgence

  • The computer is invented [to connect you and me].

不定式to connect you and me，作目的状语

分词短语

现在分词短语

doing+短语

成分：定语，状语

且动作由主语发出

  • I remember that we always anticipated [talking] with him with great eagerness because his lectures are delivered, never [failing to provoke chuckles or loud laughs]

talking动名词，作宾语

failing to provoke chuckles or loud laughs现在分词短语，作状语

过去分词

do + ed

表被动，修饰最近的名词

  • A good teacher add considerably to the materials [contained in the mobile phones]

独立主格结构

[Being 20], I meet you.

being 与主语I动作一致

[The eyes sparkling with delight], the daughter tell her mother.

想表达的sparkling与主语动作不一致，在其前加上逻辑主语The eyes

members

members include attributes(all types of the variables) and method(the functions that belongs to the class).

Access specifier

Access specifiers define how the members (attributes and methods) of a class can be accessed.

  • public - members are accessible from outside the class
  • private - members cannot be accessed (or viewed) from outside the class
  • protected - members cannot be accessed from outside the class, however, they can be accessed in inherited classes.

Note: By default, all members of a class are private if you don’t specify an access specifier:

Encapsulation

If you want others to read or modify the value of a private member, you can provide public get and set methods.

Inheritance

In C++, it is possible to inherit attributes and methods from one class to another.

  • We can also use multiple inheritance

Polymorphism

Just override the method from the base class in the derived class.

Expectations

When an error occurs, C++ will normally stop and generate an error message. The technical term for this is: C++ will throw an exception (throw an error).

  • Exception handling in C++ consist of three keywords: try, throw and catch:

    if a error is detected in the try{}, throw the exception and handle it in catch{}.

logical operators

  • ||, &&, !, which treat any nonzero argument as representing TRUE and argument 0 as representing FALSE. And then they return either 1 or 0, indicating a result of either TRUE or FALSE, respectively.

  • A second important distinction between the logical operators ‘&&’ and ‘||’
    versus their bit-level counterparts ‘&’ and ‘|’ is that the logical operators do not
    evaluate their second argument if the result of the expression can be determined
    by evaluating the first argument.

unsigned and two’s complement

Fatal

Some possibly nonintuitive behavior arises due to C’s handing of expressions containing combinations of signed and unsigned quantities.

*When an operation is preformed where one operand is signed and the other is unsigned, C implicitly casts the signed argument to unsigned. *

As an example, Since sizeof would return a size_t type, the int type of maxbytes will be implicitly casts to be unsinged. Hence no matter how small the maxbytes is, the condition all way is true and call the function memcpy().

So we can rewrite the condition to be maxbytes >= sizeof(val)
